Netdata Cloud lets you monitor the health and performance of an entire infrastructure from a single interface. By
aggregating health entries and alarm statuses from each of your claimed nodes, Netdata Cloud puts distributed health
monitoring in one easy-to-use place. 

View more about each of Netdata Cloud's health monitoring features below.

    href="/docs/cloud/monitor/alarms" 
    title="Audit active alarms">
    Track the health of your infrastructure in one place by taking advantage of the powerful health monitoring watchdog running on every node.
  </DiscoverBox>
  <DiscoverBox
