# APC UPS collector

Monitors different APC UPS models and retrieves status information using `apcaccess` tool.

## Configuration

If using [our official native DEB/RPM packages](/docs/installation/native-linux-distribution-packages), make sure `netdata-plugin-chartsd` is installed.

Edit the `charts.d/apcupsd.conf` configuration file using `edit-config` from the Netdata [config
directory](/docs/configuring/configuration), which is typically at `/etc/netdata`.

```bash
cd /etc/netdata   # Replace this path with your Netdata config directory, if different
sudo ./edit-config charts.d/apcupsd.conf
```
