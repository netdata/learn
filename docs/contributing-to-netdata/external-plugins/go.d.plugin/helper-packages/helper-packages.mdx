# Helper Packages

- if you need IP ranges consider to
  use [`iprange`](/docs/contributing-to-netdata/external-plugins/go.d.plugin/helper-packages/iprange).
- if you parse an application log files, then [`log`](https://github.com/netdata/go.d.plugin/tree/master/pkg/logs) is
  handy.
- if you need filtering
  check [`matcher`](/docs/contributing-to-netdata/external-plugins/go.d.plugin/helper-packages/matcher).
- if you collect metrics from an HTTP endpoint use [`web`](https://github.com/netdata/go.d.plugin/tree/master/pkg/web).
- if you collect metrics from a prometheus endpoint,
  then [`prometheus`](https://github.com/netdata/go.d.plugin/tree/master/pkg/prometheus)
  and [`web`](/docs/contributing-to-netdata/external-plugins/go.d.plugin/helper-packages/web) is what you need.
- [`tlscfg`](/docs/contributing-to-netdata/external-plugins/go.d.plugin/helper-packages/tlscfg) provides TLS support.
- [`stm`](/docs/contributing-to-netdata/external-plugins/go.d.plugin/helper-packages/stm) helps you to convert any struct to a `map[string]int64`.
